# Vercel v0 UI/UX Design Prompt

**Copy and paste everything below this line into Vercel v0:**

---

**Project Overview:**
Design an "Executive Dashboard" for a Competitive Intelligence AI Agent. This web application takes a tech startup's name as input and runs a cluster of autonomous AI agents to perform deep, institutional-grade investment research (News, Competitors, Business Model, Risks, Opportunities). 

**Target Audience:**
Venture Capital investors, executive judges, and financial analysts. The design MUST feel like a premium, high-end Bloomberg Terminal alternative for the modern AI era. It must be brutally objective, hyper-clean, and easy to skim during a fast-paced pitch. **Do not use generic SaaS aesthetics or playful elements. Avoid a "generative AI chat" look. This is a rigid, data-driven dashboard.**

**Aesthetics & Design Language:**
- **Theme:** Ultra-sleek Dark Mode. Use a deep, space-black or very dark slate background.
- **Style:** Premium Glassmorphism. Cards should have subtle translucent backgrounds (backdrop-blur) with delicate, low-opacity borders to create a layered sense of depth.
- **Accents:** Use vibrant neon accents sparingly to draw the eye to critical data (e.g., Emerald Green for INVEST, Amber for PARTNER, Crimson Red for AVOID, Violet for WATCH).
- **Typography:** Use a bold, modern sans-serif for headings (like Outfit or Inter) and a highly legible sans-serif for body text. 
- **Layout Philosophy:** Bottom-Line-Up-Front (BLUF). The most critical information must sit at the absolute top of the screen.

**Dashboard Layout Requirements (Top to Bottom):**

**1. The Header & Input Section:**
- A clean, minimalist header reading "Competitive Intelligence Agent".
- A centered search bar (e.g., "Analyze Startup...") with a sleek glowing submit button.

**2. The Hero Section: "The Final Recommendation" (Full Width)**
- A massive, prominent card spanning the full width at the top.
- **Left Panel (The Verdict):** A large, colored badge displaying the final verdict (e.g., "INVEST", "WATCH", "AVOID"). Below it, a circular "Confidence Meter" (e.g., 7/10) with the ring color matching the verdict badge.
- **Right Panel (The TL;DR):** Text blocks for "Key Decision Drivers", "Strongest Supporting Evidence", "Core Trade-off", and "Reasoning". 
- *Crucial UX detail:* In the text blocks, monetary values (like $7.4B) and percentages (like 65%) should be heavily bolded in bright white so an executive can skim the numbers instantly.

**3. The Visual Data Row (3 Columns)**
Below the Hero Section, a row of three widget cards:
- **Card 1 (Data Quality):** A visually striking Donut Chart breaking down the trust-tiers of the sources used (Tier 1 to Tier 4). Use vibrant colors for the donut segments.
- **Card 2 (Risk Analysis):** Horizontal progress bars detailing specific risks (e.g., Regulatory, Competition). Include a severity badge (High, Medium, Low) next to each.
- **Card 3 (Growth Signals):** Horizontal progress bars (similar to Risk) showing positive growth vectors (e.g., Capital Raise, Ecosystem Expansion).

**4. The Raw Analysis Appendix (3 Columns or Masonry Grid)**
The bottom row contains dense text cards generated by the individual sub-agents. Let these cards expand naturally in height (no inner scrolling).
- **Cards:** "Recent News", "Competitive Position", "Business Model".
- **Special UI Elements in these cards:**
  - If a source is unverified (like Reddit/Twitter), display a prominent red warning badge (e.g., "⚠ 3 Unverified Citations").
  - If a fact is older than 6 months, display a small yellow pill badge inline saying "(HISTORICAL)".

**Overall Goal:** 
Generate a React/Tailwind implementation of this dashboard that looks incredibly polished, highly structured, and ready to present to a board of directors. Ensure the spacing (padding/margins) is highly generous to prevent the dense text from feeling overwhelming.
